Card Template

Simple, Pretty, Documented
  • Just the Basics

    This template is made using plain old HTML and CSS. That means, while it's not so great for bigger and more complex websites, it's great for the basic use cases.

  • On any Device

    Because it's possible a significant amount of people visiting your website will use a mobile device, this template is completely responsive, which means it looks great on any screen size.

  • Easy to Learn

    If you're new to web development, this template is fully documented, which hopefully means it's really easy to jump in and learn while you make it your own.

  • No Strings

    This template is completely free and open source. It's released under the MIT License, which means you can use, modify and distribute your own version without restrictions.

  • Grab and Go

    If you sign up for your own GitHub account, you can fork this project and use GitHub Pages to get your own version online in a matter of minutes.

  • Made with Love

    This project was created for an awesome friend to help her get started making her own content online. I'd love to hear from you if you found this template useful.

Resources

This template uses some great resources that you should definitely check out.

  • Ionicons

    Functional, well-designed icons that are really easy to use and also released under the MIT License.

  • Unsplash

    A great collection of high quality, free-to-use photos provided by fantastic communities around the world.

  • Flexbox Card Grid

    A great reference by Craig Anthony that I used when setting up the responsive cards in this template.
